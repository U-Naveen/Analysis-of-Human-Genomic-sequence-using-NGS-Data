# Analysis-of-Human-Genomic-sequence-using-NGS-Data
Human genomic data is extremely large and complex, containing billions of DNA base pairs. Traditional computational methods rely on handcrafted features and predefined rules, which often fail to capture complex biological patterns.

Deep learning allows models to automatically learn features directly from raw DNA sequences, improving prediction accuracy and enabling large-scale genomic analysis.

This project implements a deep learning-based genomic sequence classifier that processes DNA sequences and predicts their class using a CNN architecture.
